## ADDED Requirements

### Requirement: 会话必须归属于具体用户
系统 SHALL 将每个聊天会话绑定到创建它的具体用户。

#### Scenario: 用户创建新会话
- **WHEN** 已登录用户创建一个新会话
- **THEN** 系统将该会话归属到当前用户名下

### Requirement: 用户只能看到自己的会话列表
系统 SHALL 仅返回当前登录用户自己的会话列表。

#### Scenario: 查询会话列表
- **WHEN** 已登录用户请求会话列表
- **THEN** 系统只返回该用户拥有的会话

### Requirement: 用户不能访问其他用户的消息
系统 SHALL 阻止一个用户访问不属于自己的会话和消息。

#### Scenario: 访问其他用户会话
- **WHEN** 用户请求一个不属于自己的会话消息
- **THEN** 系统拒绝访问并返回无权限或不存在错误
