## ADDED Requirements

### Requirement: 新用户拥有默认头像
系统 SHALL 在新用户注册成功后为其分配默认头像。

#### Scenario: 创建新用户资料
- **WHEN** 用户注册成功
- **THEN** 系统为该用户创建资料，并填入默认头像信息

### Requirement: 用户可以查看自己的资料
系统 SHALL 允许已登录用户获取自己的用户名、账号和头像资料。

#### Scenario: 获取当前用户资料
- **WHEN** 已登录用户请求当前用户资料
- **THEN** 系统返回该用户的基本资料信息

### Requirement: 用户可以更换头像
系统 SHALL 允许已登录用户更新自己的头像信息。

#### Scenario: 更新头像成功
- **WHEN** 已登录用户提交一个合法的头像值
- **THEN** 系统保存新的头像信息并在后续资料查询中返回更新后的结果
