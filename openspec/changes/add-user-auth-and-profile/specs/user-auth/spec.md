## ADDED Requirements

### Requirement: 系统支持用户注册
系统 SHALL 允许新用户使用账号、密码和用户名完成注册。

#### Scenario: 注册成功
- **WHEN** 用户提交一个未被占用的账号、合法密码和用户名
- **THEN** 系统创建新用户并返回成功结果

#### Scenario: 账号已存在
- **WHEN** 用户提交一个已被占用的账号进行注册
- **THEN** 系统拒绝注册并返回账号已存在的错误

### Requirement: 系统支持用户登录
系统 SHALL 允许已注册用户使用账号和密码登录系统。

#### Scenario: 登录成功
- **WHEN** 用户提交正确的账号和密码
- **THEN** 系统返回登录成功结果，并建立后续请求可识别的登录态

#### Scenario: 密码错误
- **WHEN** 用户提交存在的账号但密码错误
- **THEN** 系统拒绝登录并返回认证失败错误

### Requirement: 系统不会以明文方式存储密码
系统 SHALL 以密码摘要形式保存用户密码，而不是以明文形式保存。

#### Scenario: 保存新用户密码
- **WHEN** 系统创建新用户
- **THEN** 系统保存密码摘要，而不是原始密码文本
