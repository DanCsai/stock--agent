## MODIFIED Requirements

### Requirement: 系统提供助手问答 API
后端 SHALL 提供一个接收会话标识和用户问题的 API，用于执行助手问答流程、识别基金查询意图，并返回结构化基金结果或普通回答结果。

#### Scenario: 成功执行基金查询问答
- **WHEN** 已登录客户端针对一个属于当前用户的会话提交基金查询问题
- **THEN** 后端返回聊天文本、结构化基金卡片数据、走势数据以及推荐/分析占位数据

#### Scenario: 使用未知或无权限会话发起问答
- **WHEN** 客户端提交的会话标识不存在或不属于当前登录用户
- **THEN** 后端返回一个表示会话无效或无权限的错误

### Requirement: API 响应支持前端聊天渲染
后端 SHALL 返回足够完整的结构化数据，使前端可以渲染聊天文本、基金卡片、走势区域和占位区。

#### Scenario: 发送基金查询后渲染结果
- **WHEN** 前端收到基金查询问答接口的响应
- **THEN** 前端可以基于返回数据直接渲染聊天文本、基金卡片和走势图
