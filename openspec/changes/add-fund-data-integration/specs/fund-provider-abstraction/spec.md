## ADDED Requirements

### Requirement: 系统提供独立的基金数据服务抽象
系统 SHALL 将基金搜索、详情和走势能力封装为独立服务，而不是绑定到网页入口逻辑中。

#### Scenario: 网页端调用基金服务
- **WHEN** 网页端需要执行基金搜索或查询
- **THEN** 它通过统一基金服务能力完成请求，而不是直接调用外部数据源

### Requirement: 基金数据能力可被多入口复用
系统 SHALL 使基金查询能力可被网页端和后续飞书机器人共同复用。

#### Scenario: 接入新入口
- **WHEN** 后续新增飞书机器人等入口
- **THEN** 新入口可以复用同一套基金服务，而无需重写基金查询逻辑
