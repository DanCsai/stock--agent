## ADDED Requirements

### Requirement: 系统支持基金基础信息查询
系统 SHALL 返回基金的基础信息，用于卡片展示。

#### Scenario: 查询基金详情
- **WHEN** 用户选定一只基金进行详情查询
- **THEN** 系统返回基金名称、代码、类型、基金公司、最新净值和日涨跌信息

### Requirement: 查询结果可被结构化展示
系统 SHALL 以结构化字段返回基金详情，而不是仅返回纯文本描述。

#### Scenario: 前端渲染基金卡片
- **WHEN** 前端收到基金详情结果
- **THEN** 前端可以直接使用结构化字段渲染基金卡片
